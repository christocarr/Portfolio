Netflix Clone

capstone project

Learned about hooks, compound components, and styled components. I especially love the context API that allows the developer to add state to any component with out having to pass it down to child components, know as props drilling. React router is used for declarative routing making this clone a single page application.

Authentication and user accounts is managed by Firebase so you can create an account and log in as it were the real Netflix.
