personal project

A React create, read, update and delete (CRUD) app.

This app was built to teach myself Redux; the tiny but powerful state management library. By combining Redux with React allows React developers to easily manage state that is shared across components. This is especially true with trying to develop apps with complex application state. Redux can be used with any JavaScript framework or library even vanilla JS.
