About me

frontend developer based in london

I'm an ambitious self-taught frontend web developer. I specialize in frontend development and love creating great user interactions and things for the web.

I started my self-taught web development journey when I built my first website over ten years ago - a photography portfolio. So much has changed since then. In 2017, to find out if I still enjoyed web development, I took part in Udacity's Google Developer Challenge Scholarship which I finished and then was given a chance to take part in the full Google Developer Nanodegree Scholarship, which I successfully completed.

Since then I have not looked back and continue honing my skills. I love using React and the latest technologies, especially the JAMStack, to build modern websites and apps. I also study towards a BSc in Computing and IT as a distance learning student at The Open University.
