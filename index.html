Who I am

web developer based in london

I'm an ambitious self-taught web developer working in London, UK. I specialize in frontend development and love creating great user interactions and things for the web.

I looked into web development as career when I built my photography website by teaching myself the basics of HTML and CSS. To find out if I still enjoyed web development, I took part in Udacity's Google Developer Challenge Scholarship which I finished and then successfully completed a full six month Google Developer Nanodegree Scholarship. Since then I have not looked back and enjoyed every minute.

I love using React and the latest technologies to build modern websites and apps. While not freelancing and working as a web developer I study towards a BSc in Computing and IT as a distance learning student at The Open University.
